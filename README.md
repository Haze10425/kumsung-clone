- **프로젝트 사이트:** [바로가기](https://haze10425.github.io/kumsung-clone/)

---
## 기술 스택

- **기술 스택:** React, TypeScript, SCSS, GSAP, Swiper

---

## 핵심 기능
- 스크롤 트리거를 활용한 **한 섹션 단위 스크롤** 구현
- 상단 이동 버튼 및 스크롤 상태에 따른 UI 변경
- 재사용 가능한 **슬라이드 컴포넌트** 구현 (Splide + AutoScroll)
- 금성교과서 프로젝트 섹션 **그래프/라인 시각화** 구현 및 반응형 적용
- Tag Cloud 라이브러리를 이용한 **단어 클라우드 시각화**
- 모바일/데스크탑 반응형 레이아웃 적용

---

## 기술적 특징
- React 컴포넌트 분리 및 재사용 구조 설계
- SCSS 변수와 mixin으로 **공용 색상, 폰트, 반응형 브레이크포인트 관리**
- GSAP ScrollTrigger를 활용한 **애니메이션 및 스크롤 이벤트 처리**
- Swiper 옵션 활용으로 **무한 루프 슬라이드, 반응형, 속도 조절 구현**
- 그래프와 슬라이드, 태그 클라우드 모두 **반응형 적용**
- 웹 표준과 접근성 고려
